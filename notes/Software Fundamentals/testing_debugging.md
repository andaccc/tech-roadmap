# Testing and Debugging
## Static Analysis : without execute the code
- Security vulnerabilities
- Bug risks and anti-patterns (null/undefined values?)
- Code style 
- Performance issues
- Unused code

## Dynamic Analysis : execute the code
- Unit test
- Integration test
- System test
- Acceptance test

## Testing Process:
- Manual Testing
- Automation Testing
- Functional 
  - Smoke Testing
    - Simple validation to checking working or not
  - Unit test
  - Integration test
  - System test
  - Regression test
    - new changes
  - Acceptance test
- Non-functional
  - Performanance
    - Load/Stress test
  - Usability 
  - Security
